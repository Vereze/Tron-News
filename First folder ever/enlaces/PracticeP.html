Como hacer una pagina con html de forma basica

Como empezar:

Para empezar hay que conocer las etiquetas <p>, de heading y listo eso seria lo mas basico para programar una pagina.

Ya con estas etiquetas podemos hacer una pagina, en la que usaremos las de heading para hacer un titulo, y <p> para los parrafos a escribir. Tal y como se puede ver aqui mismo.

Ya despues podremos ver cosas como los botones y darle una forma con CSS.

Desde aqui ya se puede ir viendo otras etiquetas:
  • <em> : Para darle enfasis, lo pone en Cursiva.
  • <strong> : Para poner en Negrita.
  • <br> : Da un salto entre lineas muy explicito.
  • <pre> : 
    Respeta los espacios en blanco    que dejes.

Y esas serian algunas mas que puedes aprender despues. Todas estas se usan sobre el texto del documento.

Mas adelante puedes aprender a hacer botones, listas y tablas.

Al boton podremos, mediante JS, darle una funcionalidad.

Tipos de lista

Existen tres tipos de listas:

  • Unordered lists
  • Ordered lists
  • Definition lists

Unordered lists:

Las listas desornenadas, son estas que tienen un maracdor X en cada elemento. Y son asi:

  • Elemento de lista desornenada.
  • Elemento de lista desornenada.

Ordered lists

Las listas ordenadas tienen un maracdor que muestra un cierto orden en cada elemento:

  1. Primer elemento
  2. Segundo elemento
    1. Subelemento 1
    2. Subelemento 2
  3. Tercer elemento
